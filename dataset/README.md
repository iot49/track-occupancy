# Occupancy Dataset Preparation

This directory contains the tools for extracting training data from `.r49` railroad layout archives.

## Scripts

### `pnpm run prep`
*   **Purpose:** Scans the `r49/` directory (including subfolders) for `.r49` files.
*   **Action:** Extracts 136x136 image crops for every marker defined in the archives.
*   **Splitting:** Performs a deterministic 80/20 train/validation split using a hash of the archive name, image name, and marker ID.
*   **Output:** Generates a structured image database in `db/` organized by class folders, which is consumed by the CNN training pipeline in the `cnn/` directory.

## Directory Structure
*   `r49/`: Place your source `.r49` files here.
*   `db/`: The generated training database (auto-generated by the prep script).
*   `src/data_prep.ts`: The extraction and splitting logic.
